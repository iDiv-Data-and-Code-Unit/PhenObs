Editing a collection

You may edit a collection in either “Local Observations” view or “Overview” view. You will find instructions for both options below.

Please follow the below steps for editing a collection in the Local Observations view:

  1. {% include "help/partials/helper_link.html" with id="check_connection_edit" text="Check connectivity status and verify 'Online' state." %} {% include "help/partials/check_connection.html" with id="check_connection_edit"%}
  2. Proceed to Local Observations page.
  3. Locate the collection you would like to edit.
    1. If the collection is not stored on the device, click on “All collections” to retrieve all the available collections from the database.
    2. {% include "help/partials/helper_link.html" with id="edit_col_page" text="Proceed to 'Edit' page for the given collection." %} {% include "help/partials/edit_col_page.html" with id="edit_col_page"%}
  4. {% include "help/partials/helper_link.html" with id="collection_date_edit" text="Check the collection date." %} {% include "help/partials/collection_date.html" with id="collection_date_edit"%}
  5. {% include "help/partials/helper_link.html" with id="plant_selection_edit" text="Select the plant." %} {% include "help/partials/plant_selection.html" with id="plant_selection_edit"%}
  6. {% include "help/partials/helper_link.html" with id="filling_values_edit" text="Fill in values for the chosen plant." %} {% include "help/partials/filling_values.html" with id="filling_values_edit"%}
  7. Select next plant and repeat step 6 for every record you would like to edit.
  8. {% include "help/partials/helper_link.html" with id="saving_local_obs_edit" text="Save the collection to the database." %} {% include "help/partials/saving_local.html" with id="saving_local_obs_edit"%}

Please follow the below steps for editing a collection in the Overview view:

  1. {% include "help/partials/helper_link.html" with id="check_connection_edit2" text="Check connectivity status and verify 'Online' state." %} {% include "help/partials/check_connection.html" with id="check_connection_edit2"%}
  2. Proceed to Overview page.
  3. {% include "help/partials/helper_link.html" with id="garden_selection_overview" text="Select the garden(s)." %} {% include "help/partials/garden_selection_overview.html" with id="garden_selection_overview"%}
  4. Select “Edit” view.
  5. Click on the collection you would like to edit.
  6. {% include "help/partials/helper_link.html" with id="filling_values_overview" text="Fill in values." %} {% include "help/partials/filling_values_overview.html" with id="filling_values_overview"%}
  7. {% include "help/partials/helper_link.html" with id="saving_overview" text="Save the collection." %} {% include "help/partials/saving_overview.html" with id="saving_overview"%}
