Adding a collection

You may add a collection in either “Local Observations” view or “Overview” view. You will find instructions for both options below.

Please follow the below steps for adding a collection in the Local Observations view:

  1. {% include "help/collecting/partials/helper_link.html" with id="check_connection" text="Check connectivity status and verify 'Online' state." %} {% include "help/collecting/partials/check_connection.html" with id="check_connection"%}
  2. {% include "help/collecting/partials/helper_link.html" with id="add_collection_page" text="Proceed to 'Add collection' page." %} {% include "help/collecting/partials/add_collection_page.html" with id="add_collection_page"%}
  3. {% include "help/collecting/partials/helper_link.html" with id="collection_date" text="Check the collection date." %} {% include "help/collecting/partials/collection_date.html" with id="collection_date"%}
  4. {% include "help/collecting/partials/helper_link.html" with id="subgarden_selection" text="Select the subgarden(s) you would like to enter data for." %} {% include "help/collecting/partials/subgarden_selection.html" with id="subgarden_selection"%}
  5. {% include "help/collecting/partials/helper_link.html" with id="plant_selection" text="Select the plant." %} {% include "help/collecting/partials/plant_selection.html" with id="plant_selection"%}
  6. {% include "help/collecting/partials/helper_link.html" with id="filling_values" text="Fill in values for the chosen plant." %} {% include "help/collecting/partials/filling_values.html" with id="filling_values"%}
  7. Select next plant and repeat step 6 for every plant.
  8. {% include "help/collecting/partials/helper_link.html" with id="saving_local_obs" text="Save the collection to the database." %} {% include "help/collecting/partials/saving_local.html" with id="saving_local_obs"%}

Please follow the below steps for adding a collection in the Overview view:

  1. {% include "help/collecting/partials/helper_link.html" with id="check_connection_add" text="Check connectivity status and verify 'Online' state." %} {% include "help/collecting/partials/check_connection.html" with id="check_connection_add"%}
  2. Proceed to Overview page.
  3. {% include "help/collecting/partials/helper_link.html" with id="garden_selection_overview_add" text="Select the garden(s)." %} {% include "help/collecting/partials/garden_selection_overview.html" with id="garden_selection_overview_add"%}
  4. Select “Edit” view.
  5. {% include "help/collecting/partials/helper_link.html" with id="overview_add" text="Click on 'Add a new collection'" %} {% include "help/collecting/partials/add_overview.html" with id="overview_add"%}
  6. {% include "help/collecting/partials/helper_link.html" with id="filling_values_overview_add" text="Fill in values." %} {% include "help/collecting/partials/filling_values_overview.html" with id="filling_values_overview_add"%}
  7. {% include "help/collecting/partials/helper_link.html" with id="saving_overview_add" text="Save the collection." %} {% include "help/collecting/partials/saving_overview.html" with id="saving_overview_add"%}
