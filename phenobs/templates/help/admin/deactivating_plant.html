Deactivating a plant

If you would like to deactivate an already existing plant from a subgarden to avoid it showing up in the new collections, please follow the below steps:

  1. {% include "help/admin/partials/helper_link.html" with id="plants_module3" text="Please locate the 'PLANTS' module and click on 'Plants'." %} {% include "help/admin/partials/plants_module.html" with id="plants_module3"%}
  2. {% include "help/admin/partials/helper_link.html" with id="plant_select" text="Locate the plant you want to deactivate." %} {% include "help/admin/partials/plant_select.html" with id="plant_select"%}
  3. Uncheck “Active”.
  4. Save.
