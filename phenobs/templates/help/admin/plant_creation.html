Plant creation

Creating plants via CSV file

A garden may have tens or hundreds of plants in it. Creating each and every plant by hand one by one can be an exhausting process. So, there is the option to create multiple plants at a time with the help of a CSV file that fits the below template:

  1. First column named “Order” indicating the plant’s order in the garden.
  2. Second column named “Name” indicating plant’s name used in the garden.
  3. Third column named “Subgarden” indicating which subgarden the plant belongs to.

You may download the template by clicking on this link.

Be aware, each subgarden needs to be created prior to uploading the CSV file. If the mentioned subgardens are not found in the system, you will get an error.
