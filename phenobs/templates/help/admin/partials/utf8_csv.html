To make sure data is saved and stored as intended, all the characters same as collected you need to save your CSV files as UTF-8. UTF-8 is a standard that support many characters, letters, digits and signs. Therefore it is important that you save your CSV files as UTF-8.

Older versions of Excel lack the feature to save as UTF-8 CSV and you may also run into some other problems while saving as UTF-8 CSV. In that case, please follow this link for a tutorial that fits your configuration.
