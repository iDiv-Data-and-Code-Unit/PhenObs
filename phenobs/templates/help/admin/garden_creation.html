Garden creation

Creating a main garden

Every garden structure is combined of a main garden instance and its subgardens. To create a main garden, please follow the below steps:

  1. Proceed to Admin Panel.
  2. {% include "help/admin/partials/helper_link.html" with id="gardens_module" text="Locate the 'GARDENS' module and click on 'Add' located on the right end of the 'Gardens' row." %} {% include "help/admin/partials/gardens_module.html" with id="gardens_module"%}
  3. {% include "help/admin/partials/helper_link.html" with id="garden_details" text="Enter garden details." %} {% include "help/admin/partials/garden_details.html" with id="garden_details"%}
  4. Leave the "Permissions" section empty and Save.

Creating a subgarden

To create a subgarden for your main garden, please follow the below steps:

  1. {% include "help/admin/partials/helper_link.html" with id="gardens_module2" text="Locate the 'GARDENS' module and click on 'Add' located on the right end of the 'Gardens' row." %} {% include "help/admin/partials/gardens_module.html" with id="gardens_module2"%}
  2. {% include "help/admin/partials/helper_link.html" with id="subgarden_details" text="Enter subgarden details." %} {% include "help/admin/partials/subgarden_details.html" with id="subgarden_details"%}
  3. Assign the users to their respective subgardens.
  4. Save.
